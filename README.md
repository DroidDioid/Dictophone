# Dictophone
Приложение для записи звука и просмотра записей.
Состоит из двух экранов:
* Экран записи звука.
* Экран просмотра списка записей и их прослушивания.

Пример работы приложения:

Использованные библиотеки и технологии:
* MVVM
* Data Binding
* Live Data
* Navigation
* Room
